# Student Performance Analysis

This project performs data cleaning and analysis on student marks data.

## Objectives
- Clean messy real-world data
- Handle missing and invalid values
- Perform department-wise analysis

## Analysis Performed
- Average marks per department
- Pass percentage per department
- Topper identification

## Tools Used
- Python
- Pandas
  
## Insights & Recommendations
- IT shows consistent performance with a 100% pass rate, indicating effective teaching practices.
- CSE’s performance may rely on a few high scorers, masking weaker overall performance.
- Recommendation: Analyze and replicate IT’s teaching and assessment practices in CSE.
- Recommendation: Provide targeted academic support to improve CSE student outcomes.
